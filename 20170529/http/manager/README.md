## 获取全部数据 /user  GET
## 添加数据 /user  POST
## 删除某一条数据 /user  DELETE
## 修改某一条数据 /user  PUT
## 获取一个数据 /user?id=1 GET

## 根据http动词区分，get，post，delete，put --> restful风格
### 不管增删改查都返回最新数据
### 正常，添加返回添加那一项，删除返回空对象，修改返回修改的那一项，查询返回查询的数据
